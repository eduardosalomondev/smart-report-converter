# 🚀 Conversor de Relatórios Inteligente

Sistema de automação para conversão de arquivos **TXT** em **Excel (.xlsx)** desenvolvido em **Python** com **Flask**.

---

## 📌 Sobre o projeto

Este projeto foi criado para automatizar a conversão de arquivos TXT em planilhas Excel de forma rápida, organizada e segura.

O sistema permite:

- 📂 Upload de arquivos TXT
- 📊 Conversão automática para Excel
- 👤 Sistema de Login
- 🔐 Cadastro de usuários
- 💳 Integração com Stripe
- 📈 Barra de progresso em tempo real
- 📥 Download automático do arquivo convertido
